Sibelius Tutorials Master Lists


Below is listed the keyboard shortcuts I've learnt, where different things are in the ribbon, and a few select options.

Keyboard Shortcuts Master List

  • Esc = deselect everything
  • Left Arrow = moves forwards through music one object at a time (for one instrument)
  • Arrows = up, down, left, right (for tab)
  • Forward Arrow & Back Arrow (or Space Bar) = extends crescendo or diminuendo
  • Command, Left Arrow = moves forward through music one bar at a time (for one instrument)
  • Space Bar = extend slur
  • Letters = notes
  • Numbers = note value/length
  • Full Stop = rhythm dot
  • Tab = selects first object in view
  • Tab = next bar (chord symbols)
  • Command, y = redo
  • Command, k = chord symbol
  • Command, t = technique
  • Alt, Command, t = tempo
  • Alt, Command, c = compare
  • Alt, click = paste chord symbol
  • Shift, h = diminuendo
  • Shift, t = transpose
  • Shift, number = interval above
  • Shift, number = fret number (for tab)
  • Shift = next chord (chord symbols)
  • Shift, Command, k = equivalent chord text
  • Shift, Tab = moves back one object at a time (for one instrument)
  • Shift, Alt, q = qwerty control for virtual keyboard
  • a = bottom C of octave (with virtual keyboard)
  • w = C# (with virtual keyboard)
  • s = D (with virtual keyboard)
  • e = D# (with virtual keyboard)
  • d = E (with virtual keyboard)
  • f = F (with virtual keyboard)
  • t = F# (with virtual keyboard)
  • g = G (with virtual keyboard)
  • y = G# (with virtual keyboard)
  • h = A (with virtual keyboard)
  • u = A# (with virtual keyboard)
  • j = B (with virtual keyboard)
  • k = top C of octave (with virtual keyboard)
  • z = go octave down (with virtual keyboard)
  • x = go octave up (with virtual keyboard)
  • s = add slur/tie
  • x = change stem direction
  • m = show mixer
  • t = show time signature gallery
  • w = change between parts and full score
  • l = show line gallery
  • h = crescendo

Ribbon Master List

  • Home > Clipboard > Paste > Paste As Cue
  • Home > Clipboard > Select Graphic
  • Home > Cliboard > Copy
  • Home > Clipboard > Capture Idea
  • Home > Instrument > Change
  • Home > Select > Filters > Chord Symbols
  • Note Input > Arrange > Reduce
  • Note Input > Arrange > Arrange
  • Notations > Noteheads > Type
  • Text > Chord Symbols > Chord Symbol
  • Text > Chord Symbols > Respell Chord Text
  • Layout > Hiding Staves
  • Layout > Breaks > System Breaks
  • View > Document View > Panorama
  • View > Panels > Navigator
  • View > Panels > Ideas
  • View > Invisibles > Layout Marks
  • Review > Versions
  • Keypad > Jazz Articulations (fifth panel)

Selecting Master List

  • Double click note to select full chord
  • Double click bar to select instrument’s system
  • Triple click bar to select all of the instrument’s music

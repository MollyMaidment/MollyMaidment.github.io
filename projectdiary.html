Sibelius Tutorials Diary

Sibelius Tutorials Project Diary


For each day I work on the tutorials, I'll outline what I've learnt in a simple list style. This isn't a "Dear Diary" (sorry to disappoint), this is a record that I can come back to and easily understand.

Day 1 (Tutorial 1)

Panels: In View>Panels there is a group of tools to help with different features of Sibelius. One of these is the Navigator, which shows the entire score in miniature with a smaller box showing which part of the score you are viewing.
Panorama: In View>DocumentView there is a panorama option, which shows the score in one long line.
Versions: In Review>Versions you can create, edit and view different versions of the same score. This is useful if you wat to experiment, but keep the original as well. Like parts, versions can be opened in the same window, making it easy to swap between.
Hiding & Focusing Staves: In Layout>HidingStaves there are options to show only particular staves, or remove empty staves from view.
Keyboard Shortcuts:
Esc = deselect everything
Left Arrow = moves forwards through music one object at a time (for one instrument)
Command, Left Arrow = moves forward through music one bar at a time (for one instrument)
Tab = selects first object in view
Shidt, Tab = moves back one object at a time (for one instrument)

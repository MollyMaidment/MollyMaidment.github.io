Sibelius Tutorials Diary

Sibelius Tutorials Project Diary


For each day I work on the tutorials, I'll outline what I've learnt. This is a record that I can come back to and understand, and I've done my best to make it understandable for anyone else who can use this information. For the full tutorials, go here: Sibelius 7 Tutorials

Day 1 (Tutorial 1)

Panels: In View>Panels there is a group of tools to help with different features of Sibelius. One of these is the Navigator, which shows the entire score in miniature with a smaller box showing which part of the score you are viewing.
Panorama: In View>DocumentView there is a panorama option, which shows the score in one long line.
Versions: In Review>Versions you can create, edit and view different versions of the same score. This is useful if you wat to experiment, but keep the original as well. Like parts, versions can be opened in the same window, making it easy to swap between.
Hiding & Focusing Staves: In Layout>HidingStaves there are options to show only particular staves, or remove empty staves from view.
Keyboard Shortcuts:

  • Esc = deselect everything
  • Left Arrow = moves forwards through music one object at a time (for one instrument)
  • Command, Left Arrow = moves forward through music one bar at a time (for one instrument)
  • Tab = selects first object in view
  • Shift, Tab = moves back one object at a time (for one instrument)

Day 2 (Tutorial 1)

Keyboard Shortcuts:

  • Letters = notes
  • Numbers = note value/length
  • . = rhythm dot
  • Shift, Number = interval above
  • Command, Y = redo

Virtual Keyboard:

  • shift, alt, q = qwerty control
  • a = bottom C
  • w = C#
  • s = D
  • e = D#
  • d = E
  • f = F
  • t = F#
  • g = G
  • y = G#
  • h = A
  • u = A#
  • j = B
  • k = top C
  • z = go octave down
  • x = go octave up

Day 3 (Tutorial 1)

Keyboard Shortcuts:

  • s = add slur/tie
  • x = change stem direction
  • m = show mixer
  • t = show time signature gallery

(Tutorial 2)
Photoscore Lite: A separate application that works similarly to and with Sibelius to scan, edit, and inport sheet music to Sibelius.

Day 4 (Tutorial 2)

Keyboard Shortcuts:

  • h = crescendo
  • Shift, h = diminuendo
  • Forward Arrow & Back Arrow (or Space Bar) = extends crescendo or diminuendo
  • l = show line gallery
  • Space Bar = extend slur
  • Command, t = technique
  • Shift, t = transpose
  • Alt, Command, t = tempo
  • Alt, Command, c = compare

Day 5 (Tutorial 2)

System Break: In Layout>Breaks>SystemBreak allows the music to automaticaly break at a certain point, so that music with multiple pages is easier to read.
Layout Marks: If the box Layout Marks in View>Invisibles is ticked, a small blue image like a sheet of paper appears, showing that there is a cover page. If selected and deleted, the cover page will be deleted as well.
Cues: Select the passage you want to have as a cue, then go Home>Clipboard>Paste>PasteAsCue and select where you want to paste it.
To swap between open parts and the full score, select music of the intrument you want in the score, click W, and it will view the part. Simply click W again to return to the full score.
Triple clicking selects all music for the intrument selected.
Piano Reduction: Add piano to your score, select the music you want, and go NoteInput>Arrange>Reduce. Select piano from the drop-down, or piano~(2) for the left hand.
Copying Graphics to Other Apps:

  1. Select
  2. Home>Clipboard>Copy
  3. Home>Clipboard>Copy
  4. Paste in other app (e.g. Microsoft Word)

Day 6 (Tutorial 3)

Double click note to select full chord.
Notehead Type: Notations>Noteheads>Type
Changing to Tab: Home>Instruments>Change, then choose whatever tab instrument you want (e.g. Acoustic Guitar, standard tuning [tab]).
Tab Notation:

  • Shift, number = fret number (note)
  • Arrows = up, down, left, right

Ideas (View>Panels>Ideas):

  • Press and hold ideas to play them
  • Select idea, copy and paste to use an idea
  • Select section, the go Home>Clipboard>CaptureIdea to add ideas
  • When searching ideas, click enter after clearing to view all

Repeat sign is located on fifth panel of Keypad (Jazz Articulations).

Day 7 (Tutorial 3)

Chord Symbols (Text>ChordSymbols>ChordSymbol or Command, k):

  • shift = next chord
  • tab = next bar
  • Status bar shows chord of notes selected
  • Text>ChordSymbols has multiple options for changing chord text, symbols, or removing one of the two
  • Shift, Command, k = equivalent chord text
  • Text>ChordSymbols>RespellChordText is another one of the options
  • To copy chord symbols go Home>Select>Filters>ChordSymbols then select where you want to paste it, press Alt and click

To add a backing line (backing strings for example), add the instrumetns wanted for backing, then select and copy the music you want arranged for the backing. Go NoteInput>Arrange>Arrange, select from the options (e.g. standard arrangement) and click OK.
